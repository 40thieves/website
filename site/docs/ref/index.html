Reference Manual

For the time being, CodeMirror 6 is published as a single package, @codemirror/next, of which the modules listed here are subdirectories. So to, for example, load the text module, you'd import or require "@codemirror/next/text".

(In the future, these will most likely be published as separate npm packages.)

The each package contains CommonJS and ECMAScript modules. You'll have to use some kind of bundler to run them in the browser.

The main modules you'll interact with are view, which provides the UI component for an editor, and state, which implements the data structure that tracks the editor state.

A minimal editor might look like this:

import {EditorView} from "@codemirror/next/view"
import {EditorState} from "@codemirror/next/state"

let myView = new EditorView({state: EditorState.create({doc: "hello"})})
document.body.appendChild(myView.dom)

To get functionality like highlighting, a line number gutter, or an undo history, you need to add various extensions to such a basic editor.
